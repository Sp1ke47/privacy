SZSYIDC 隐私政策

上次更新时间： 2022 年 3 月 15 日

深圳实验网络部署委员会（拉丁字母缩写及下文简称为 SZSYIDC ; 英文全称 Internet Deployment Committee of Shenzhen Experimental School）是旨在完善中国对国际网络访问环境缺陷而成立的互助性组织。SZSYIDC 将按照本隐私政策的规定保护您的个人信息，并且以高度勤勉、审慎的态度与义务对待这些信息。

本隐私政策阐述并规定了 SZSYIDC 处理的个人数据以及 SZSYIDC 处理个人数据的方式和目的。

在未征得您事先许可的情况下，委员会任何成员不会将这些信息对外披露或向第三方提供。

一、适用范围

  • 在您使用 Project Atriox（位于美国洛杉矶的 Trojan 协议境外网络访问服务）时，Project Atriox 服务器（下文以其IP地址 199.180.112.18 代称）将自动接收并记录的您的IP地址、访问日期和时间；

  • 在您使用 Project Beethoven（Microsoft 365 E5 开发者订阅）时，Microsoft Corporation 将自动接收并记录的您的浏览器和计算机上的信息，包括但不限于您的IP地址、浏览器的类型、使用的语言、访问日期和时间、软硬件特征信息及您需求的网页记录等数据。

二、信息使用

  • SZSYIDC 不会向任何无关第三方提供、出售、出租、分享或交易您的个人信息，除非事先得到您的许可，或该第三方和 SZSYIDC 单独或共同为您提供服务，且在该服务结束后，其将被禁止访问包括其以前能够访问的所有这些资料；

  • SZSYIDC 亦不允许任何第三方以任何手段收集、编辑、出售或者无偿传播您的个人信息。任何委员如从事上述活动，一经发现，SZSYIDC 有权立即褫夺其在 SZSYIDC 中的委员资格。

三、信息披露

在如下情况下，SZSYIDC 将依据您的个人意愿或法律的规定全部或部分的披露您的个人信息：

  • 经您事先同意，向第三方披露；

  • 为提供您所要求的产品和服务，而必须和第三方分享您的个人信息；

  • 根据法律的有关规定，或者行政或司法机构的要求，向第三方或者行政、司法机构披露；

  • 如您出现违反中国有关法律、法规或者本应用服务协议或相关规则的情况，需要向第三方披露；

  • 如您是适格的知识产权投诉人并已提起投诉，应被投诉人要求，向被投诉人披露，以便双方处理可能的权利纠纷；

  • 其它根据法律、法规或者网站政策规定的被 SZSYIDC 认为合适的披露。

四、信息存储和交换

  • 当使用 Project Atriox 时，SZSYIDC 收集的有关您的信息和资料将保存在 199.180.112.18 上，这些信息和资料绝不允许被传送至第三方，并且采用每三日自动删除 Warning 日志的手段严格管理其存储；

  • 当使用 Project Beethoven 时，有关您的信息和资料经由 Microsoft Corporation 管理和存储，SZSYIDC 无权干涉。详情请参见Microsoft 隐私声明。

五、信息安全

  • 199.180.112.18 配置了严格的密钥保护措施，且您经由 199.180.112.18 路由的流量均采用带有 TLS 加密的 Trojan 协议传输及保护，确保您的信息不丢失，不被滥用和变造。
    操作员（即 SZSYIDC 委员长 Spike_47）尽最大的努力维护及保障 Project Atriox 的强大安全性及优秀性能。请妥善保管您的用户名及密码信息。

  • 操作员有能力创建、修改及删除 Project Beethoven 用户账户的部分信息，但仅限于用户姓名、显示名称、账号、密码、联系方式、登陆许可、Microsoft 365 订阅许可证、OneDrive 存储空间（上限 5 TB）以及OneDrive 文件。
    操作员将以遵守本政策规定及社会公序良俗为基本原则，承诺未经当事人许可，不得擅自修改、查看 Project Beethoven 用户账户的任何信息。

尽管有上述安全措施，但同时也请您注意在信息网络上不存在“完善的安全措施”；

六、本隐私政策的更改

  • 如果决定更改隐私政策，我们会在本政策中、SZSYIDC Telegram 管理群中以及我们认为适当的位置发布这些更改，以便您了解我们如何收集、使用您的个人信息，哪些人可以访问这些信息，以及在什么情况下我们会透露这些信息；

  • SZSYIDC 保留随时修改本政策的权利，因此请经常查看。如对本政策作出重大更改，SZSYIDC 会以口头和/或书面通知的形式告知。

请您妥善保护自己的个人信息，并且仅在必要的情形下向他人提供。如您发现自己的个人信息泄露，尤其是 Project Atriox 用户名及密码发生泄露，请您立即联络 SZSYIDC 委员长，以便及时采取相应措施。
